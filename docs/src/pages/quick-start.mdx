# 快速开始

## 准备环境

Mahiro 本体需要运行在 Nodejs 上，在安装依赖时进行编译，还需具备 Python 环境。

我们推荐准备以下环境：

- Node.js **_v18+_**

- Python **_v3.8+_**

### 加速源

若从零开始，请预设好 npm 加速源：

```bash
  npm config set registry https://registry.npmmirror.com/
  npm i -g pnpm

  # sqlite 安装国内网络必备
  echo "better_sqlite3_binary_host=https://registry.npmmirror.com/-/binary/better-sqlite3" >> ~/.npmrc
  echo "better_sqlite3_binary_host_mirror=https://registry.npmmirror.com/-/binary/better-sqlite3" >> ~/.npmrc
  echo "node_sqlite3_binary_host=https://registry.npmmirror.com/-/binary/sqlite3" >> ~/.npmrc
  echo "node_sqlite3_binary_host_mirror=https://registry.npmmirror.com/-/binary/sqlite3" >> ~/.npmrc
```

## 启动

### 创建项目

我们将创建一个最简 Mahiro 项目：

```bash
  # 创建文件夹
  mkdir my-mahiro
  cd ./my-mahiro

  # 初始化项目
  pnpm init
```

安装依赖：

```bash
  pnpm add -D mahiro
```

注：在第一次安装时，`sqlite` 编译需一定时间，请耐心等待，如无法安装成功，请确保你配置了 [加速源](#加速源) 相关配置。

创建并编写 `index.ts` ：

```ts filename="index.ts"
import { Mahiro } from 'mahiro'

const run = async () => {
  const mahiro = await Mahiro.start({
    host: '127.0.0.1', // 默认 localhost
    port: 8086, // 默认 8086
    qq: 12345678,
  })

  mahiro.onGroupMessage('Hello world Group', async (data) => {
    if (data?.msg?.Content === 'Hello') {
      mahiro.sendGroupMessage({
        groupId: data.groupId,
        msg: {
          Content: 'Mahiro',
        },
      })
    }
  })

  mahiro.onFriendMessage('Hello world Friend', (data) => {
    if (data?.msg?.Content === 'Hello') {
      mahiro.sendFriendMessage({
        userId: data.userId,
        msg: {
          Content: 'Mahiro',
        },
      })
    }
  })
}

run()
```

最终目录结构如下：

```bash filename="目录结构"
 - my-mahiro
   - package.json
   - index.ts
```

启动程序：

```bash
  pnpm mahiro ./index.ts
```

### 配置群组

在 Mahiro 中，**一切均为白名单设计**，这是为了防止消息收发有非预期的情况，保证安全。

故群组功能必须配置后才可使用，打开控制台打印的 Mahiro 面板地址：`http://localhost:8098` （注意放通防火墙与安全组，运行在服务器上时，请替换为服务器公网 IP）。

在首页输入当前主账号，获得面板访问权限。
